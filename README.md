## Python

### 算術演算子の種類

演算子 | 説明 | 例 | 例の表示結果
---|---|---|---
+ | 足し算(加算) | print(10 + 10) | 20
- | 引き算(減算) | print(10 - 1) | 9
* | 掛け算(乗算) | print(2 * 3) | 6
/ | 割り算(除算) | print(7 / 4) | 1.75
// | 割り算の商 | print(7 // 4) | 1
% | 割り算の余り | print(7 % 4) | 3
** | べき乗(累乗) | print(2 ** 3) | 8
