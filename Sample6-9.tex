\frametitle{2次方程式の解の公式}
2次方程式$ax^2 + bx + c = 0 \,(a\neq 0)$の解は、以下の解の公式で求められる! \pause
\begin{block}{2次方程式の解の公式}
\[ x = \frac{-b \pm \sqrt{b^2-4ac}}{2a} \]
\end{block} \pause

※2次方程式では$a\neq 0$なので、解の分母が0になってしまうことはない。
